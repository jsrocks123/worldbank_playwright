## Installation

On macOS and Linux:

```sh
$ pip install poetry
$ poetry shell
$ poetry install
```

On Windows:

```sh
PS> python -m pip install worldbank-playwright
```

## Execution / Usage

To run worldbank-playwright, fire up a terminal window and run the following command:

```sh
$ pytest
```
